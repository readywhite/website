# SOP: Quote Process

## Goal

Move photo-ready leads from review to quote quickly and accurately.

## Owner

COO

## Trigger

Opportunity enters **Photos Received**.

## Steps

1. Review property address, property type, service needed, timeline, vacancy status, photos, and project notes.
2. Request clarification from SDR if scope is incomplete.
3. Prepare or coordinate the quote.
4. Move the opportunity to **Quote Sent** once the quote is sent.
5. Confirm the SDR has a follow-up task.
6. Add quote notes, assumptions, exclusions, and timing details.

## Done Definition

- Quote is sent.
- Opportunity is in **Quote Sent**.
- SDR follow-up is scheduled.
- Notes are clear enough for another operator to understand the quote.
