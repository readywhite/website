Get My Property Quote

This branded form uploads one straight-on photo per wall to a secure server-side API route for OpenAI vision analysis, deterministic package pricing, and GoHighLevel lead capture without exposing credentials in the browser.

Integration status: this site submits photos to /api/photo-estimate, then sends the standardized lead and estimate payload to /api/ghl-lead when deployed. Add OpenAI and GoHighLevel variables server-side only.

Required: tape one 8.5" x 11" paper sheet flat in portrait orientation on each wall, then upload one straight-on photo per wall. Mobile tip: keep one wall only in frame, include the full paper, avoid glare, and retake if lighting is dark.
