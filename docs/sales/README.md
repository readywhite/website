# Sales Documentation

This folder defines Ready White's sales operating system for fast lead response, package-based quoting, property-manager relationships, and recurring turnover work.

## Sales goals

- Improve speed-to-lead and close rate.
- Convert one-off repaint jobs into recurring property-management relationships.
- Use standardized packages to quote quickly and consistently.
- Use follow-up automation to prevent stale opportunities.
- Build a review flywheel after completed work.

## Target customer segments

- Property managers.
- Real estate investors.
- Apartment operators.
- Rental portfolio owners.
- Turnover coordinators.
- Agents preparing vacant listings or rentals.

## Sales positioning

Ready White is not a traditional painter. Ready White is a fast-turn property-refresh operating system built for vacancy reduction, standardized scopes, and subcontractor fulfillment.

Core message:

- Fast quote from photos.
- Standardized packages.
- Vendor-ready scopes.
- Reliable occupancy repainting.
- Designed for recurring property-management turnover.

## Follow-up priorities

- Respond immediately to new leads.
- Request missing photos before estimating.
- Follow up on sent quotes until won, lost, or disqualified.
- Recover missed calls with text-back workflows.
- Prioritize ASAP, vacant, and property-management leads.
- Ask completed customers for reviews and referrals.

## KPI scaffold

Track:

- Speed-to-lead.
- Photo completion rate.
- Quote turnaround time.
- Quote-to-approval rate.
- Follow-up completion rate.
- Closed-won rate by source and vertical.
- Recurring property-manager accounts.
- Review request conversion.
