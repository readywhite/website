# SOP: Job Closeout

## Goal

Close each job cleanly, preserve operational details, and create review/referral opportunities.

## Owner

COO

## Trigger

Opportunity enters **Completed**.

## Steps

1. Confirm work is completed.
2. Add completion notes and photos if available.
3. Confirm customer/property manager satisfaction.
4. Send review/referral request.
5. Move the opportunity to **Closed Won** after admin completion.
6. Add any follow-on property opportunities as new opportunities.

## Done Definition

- Opportunity is Closed Won.
- Completion notes exist.
- Review/referral request sent.
- Any next property opportunity is captured.
