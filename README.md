# Ready White Website

Static landing page for Ready White's property refresh painting offer.

## Launch Checklist

1. Upload the kitchen hero image to `assets/kitchen-hero.jpg`.
2. Build the GoHighLevel form named **Ready White Quote Request** using `GHL_SETUP.md`.
3. Paste the GoHighLevel Form Builder embed into the quote section of `index.html`.
4. Set the form redirect to `thank-you.html` or use the approved popup message.
5. Publish the site and test a full lead submission.

## Local Preview

```bash
python3 -m http.server 8080
```

Then open <http://localhost:8080>.
